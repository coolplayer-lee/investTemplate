## V5.5.3 核心变更

### 1. 市场周期判定标准（第六章6.0.6节新增）

**3维度判定**：
- 恒生指数52周高低点位置（>30%反弹为牛市）
- 成交量变化
- 破净个股比例（>30%为熊市信号）

### 2. 差异化安全边际标准

| 指标 | 牛市标准 | 熊市标准 |
|------|----------|----------|
| **剔除净现金PE** | < 10倍 | < 6倍 |
| **股息率** | > 3.8% | > 6% |
| **PB** | < 1.0倍 | < 0.6倍 |
| **FCF/市值** | > 8% | > 12% |
| **净现金/市值** | > 30% | > 50% |

### 3. 动态调整执行流程

```
月度重估：
├── 市场上涨>10% → 收紧标准（向熊市标准靠拢）
├── 市场下跌>10% → 放宽标准（向牛市标准靠拢）
└── 始终保持与市场估值水平匹配的安全边际
```

### 4. 牛市特别风险提示

**4类牛市陷阱**：
- "这次不一样"思维 → 坚持标准，不追涨
- 降低标准买入 → 牛市末期买入=站岗
- 忽视股息率 → 牛市中股息率<3%风险极高
- 忘记动态退出 → 牛市中更应严格执行退出纪律

## 实战应用（2026年3月港股）

**市场状态**：
- 恒指约24,000点（较2024年低点反弹约30%）
- 判定：**牛市/震荡市偏牛**
- 执行标准：**剔除净现金PE < 10倍，股息率 > 3.8%**

**标的适配**：
| 标的 | 剔除净现金PE | 股息率 | 结论 |
|------|-------------|--------|------|
| **保利物业** | ~9倍 | 4.5% | ✅ **符合，可建仓** |
| **蒙牛** | ~8倍 | 3.5% | ⚠️ **股息略低，可接受** |
| **建行H** | ~4倍 | 7%+ | ✅ **极度符合，重仓** |
| **海底捞** | >15倍 | 0% | ❌ **不符合，观望** |
